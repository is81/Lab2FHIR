## 二、技术栈

```
前端：Vue 3 + Element Plus + Vue Router + PDF.js + Axios
```

## 四、API 端点

| 端点 | 方法 | 功能 |
|------|:--:|------|
| `/api/convert` | POST | 上传 PDF → 解析 → 入库 |
| `/api/reports` | GET | 搜索+筛选+分页 |
| `/api/reports/{id}` | GET | 报告详情 |
| `/api/reports/{id}/pdf` | GET | PDF 文件 |
| `/api/stats` | GET | 统计（缓存） |
| `/api/patients/{name}/reports` | GET | 患者报告 |
| `/api/health` | GET | 健康检查 |
