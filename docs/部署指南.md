# Lab2FHIR 部署指南

> Windows Server 2019 · 手动部署 · 无需 Docker · D 盘

---

## 部署流程

```
联网电脑（开发机）                    服务器（Windows Server 2019）
      │                                        │
      │ 下载安装包                              │
      ├─ Python 安装包 ────── U盘 ──────────→ ① 安装 Python
      ├─ 部署包 Lab2FHIR/ ─── U盘 ──────────→ ② 拷贝到 D 盘
      ├─ nginx ───────────── U盘 ──────────→ ③ 解压到 D 盘
      ├─ nssm ────────────── U盘 ──────────→ ③ 解压
      │                                       ④ pip install
      │                                       ⑤ 配置 .env
      │                                       ⑥ 初始化数据库 + 创建用户
      │                                       ⑦ 注册服务 + 开机自启
```

---

## U 盘文件清单

| 文件 | 说明 |
|------|------|
| `python-3.11.9-amd64.exe` | Python 3.11 |
| `nginx-1.26.2.zip` | Web 服务器 |
| `nssm-2.24.zip` | Windows 服务注册 |
| `Lab2FHIR/` | 项目文件（前端已构建好） |
| `pip-packages/` | 离线依赖包（可选） |

> 前端已构建在 `Lab2FHIR/frontend/dist/`，服务器**不需要 Node.js**。

---

## 第一步：安装 Python

双击 `python-3.11.9-amd64.exe`。

⚠️ **勾选 "Add Python to PATH"**，然后 Install Now。

```powershell
python --version   # 验证
```

---

## 第二步：解压 nginx 和 nssm

```powershell
Expand-Archive D:\nginx-1.26.2.zip -DestinationPath D:\
Expand-Archive D:\nssm-2.24.zip -DestinationPath D:\nssm
copy D:\nssm\nssm-2.24\win64\nssm.exe C:\Windows\System32\
```

---

## 第三步：拷贝项目

```powershell
Copy-Item -Recurse D:\Lab2FHIR D:\Lab2FHIR
cd D:\Lab2FHIR
```

---

## 第四步：安装 Python 依赖

```powershell
cd D:\Lab2FHIR\backend

# 有网：
pip install -r requirements.txt

# 完全离线：
pip install --no-index --find-links=D:\pip-packages -r requirements.txt
```

---

## 第五步：配置 nginx

```powershell
Copy-Item D:\Lab2FHIR\nginx.conf D:\nginx\conf\nginx.conf -Force
```

---

## 第六步：配置密钥和密码

```powershell
cd D:\Lab2FHIR

# 生成随机密钥
$jwtSecret = -join ((48..57) + (97..102) | Get-Random -Count 64 | ForEach-Object { [char]$_ })
Write-Host "密钥: $jwtSecret"
```

复制输出的字符串，创建 `.env`：

```powershell
@"
LAB2FHIR_JWT_SECRET=粘贴密钥
LAB2FHIR_ADMIN_PASSWORD=管理员密码
LAB2FHIR_DOCTOR_PASSWORD=医生密码
CORS_ORIGINS=*
"@ | Out-File -FilePath .env -Encoding UTF8
```

---

## 第七步：初始化数据库

```powershell
cd D:\Lab2FHIR\backend
$env:PYTHONIOENCODING = "utf-8"
$env:LAB2FHIR_ENV = "development"
python tools/seed_users.py
```

记下输出的账户和密码。

---

## 第八步：注册 Windows 服务

```powershell
# 先找到 Python 确切路径
where python
# 记录输出路径，例如 C:\Users\Administrator\AppData\Local\Programs\Python\Python311\python.exe

# 注册后端服务（把 python.exe 路径换成上面查到的）
nssm install Lab2FHIR-Backend "C:\Users\Administrator\AppData\Local\Programs\Python\Python311\python.exe" "-m uvicorn backend.src.api.main:app --host 0.0.0.0 --port 8000"
nssm set Lab2FHIR-Backend AppDirectory "D:\Lab2FHIR"
nssm set Lab2FHIR-Backend AppEnvironmentExtra "PYTHONIOENCODING=utf-8"

# 注册 nginx 服务
nssm install Lab2FHIR-Nginx "D:\nginx\nginx.exe"
nssm set Lab2FHIR-Nginx AppDirectory "D:\nginx"

# 启动
nssm start Lab2FHIR-Backend
nssm start Lab2FHIR-Nginx
```

---

## 第九步：防火墙

```powershell
New-NetFirewallRule -DisplayName "HTTP 80" -Direction Inbound -Protocol TCP -LocalPort 80 -Action Allow
```

浏览器打开：**`http://服务器IP`**

---

## 日常管理

```powershell
nssm status Lab2FHIR-Backend    # 查看状态
nssm restart Lab2FHIR-Backend   # 重启
nssm status Lab2FHIR-Nginx
```

## 忘记密码

```powershell
cd D:\Lab2FHIR\backend
$env:PYTHONIOENCODING = "utf-8"
python -c "
import sys; sys.path.insert(0, 'src')
from src.db.models import SessionLocal
from src.db.user_models import User
from src.auth.security import hash_password
db = SessionLocal()
user = db.query(User).filter(User.username == 'admin').first()
user.password_hash = hash_password('新密码')
db.commit(); db.close()
print('OK')
"
```

## 更新版本

在联网电脑拉取最新代码 → 重新构建前端 → 打包 `deploy/Lab2FHIR/` → U 盘。

服务器上：

```powershell
nssm stop Lab2FHIR-Backend
# 覆盖 D:\Lab2FHIR 文件...
nssm start Lab2FHIR-Backend
```

## 故障排查

| 问题 | 解决 |
|------|------|
| `python` 不识别 | 重装 Python，勾选 Add to PATH |
| 浏览器打不开 | `Get-NetFirewallRule -DisplayName "HTTP 80"` |
| nginx 启动失败 | 查看 `D:\nginx\logs\error.log` |
| 中文乱码 | 检查 `$env:PYTHONIOENCODING = "utf-8"` |
