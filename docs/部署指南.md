# Lab2FHIR 部署指南

> Windows Server 2019 · Docker · IP 访问

---

## 准备工作

| 项目 | 说明 |
|------|------|
| Windows Server 2019 | 建议 2核4G 以上 |
| 服务器 IP 地址 | 如 `123.456.789.0` |
| 远程桌面 | Win+R → `mstsc` → 输入 IP 连接 |

---

## 第一步：安装 Docker

### 1.1 启用容器功能

右键开始菜单 → **Windows PowerShell（管理员）**，逐行执行：

```powershell
# 安装 Docker 所需的 Windows 功能
Install-WindowsFeature -Name Containers
Restart-Computer -Force
```

服务器重启后，重新远程桌面连接。

### 1.2 安装 Docker Desktop

打开浏览器，下载 Docker Desktop for Windows：

```
https://www.docker.com/products/docker-desktop/
```

双击安装包，一路"下一步"，**勾选 "Use WSL 2 instead of Hyper-V" 的话请取消**（Server 2019 不支持 WSL 2，用 Hyper-V 即可）。

安装完成后重启。

### 1.3 验证安装

打开 PowerShell，输入：

```powershell
docker --version
```

看到 `Docker version 27.x.x` 就成功了。

### 1.4 切换为 Linux 容器

右下角系统托盘 → 右键 Docker 图标 → **Switch to Linux containers**（如果已经是 Linux 容器则跳过）。

## 第二步：下载项目

打开 PowerShell：

```powershell
# 如果没有 git，先下载安装
# https://git-scm.com/download/win

cd C:\
git clone https://github.com/is81/Lab2FHIR.git
cd C:\Lab2FHIR
```

## 第三步：配置密钥和密码

**这是最重要的一步！** 在 PowerShell 中执行：

```powershell
# 生成随机 JWT 密钥
$jwtSecret = -join ((48..57) + (97..102) | Get-Random -Count 64 | ForEach-Object { [char]$_ })
Write-Host "你的 JWT 密钥: $jwtSecret"
```

复制输出的字符串（如 `a1b2c3d4e5f6...`），然后：

```powershell
# 创建 .env 配置文件（把下面三行的值换成你自己的）
@"
LAB2FHIR_JWT_SECRET=把你生成的随机字符串粘贴到这里
LAB2FHIR_ADMIN_PASSWORD=设置一个管理员密码
LAB2FHIR_DOCTOR_PASSWORD=设置一个医生密码
CORS_ORIGINS=*
"@ | Out-File -FilePath .env -Encoding UTF8
```

> ⚠️ 三处必须修改：JWT 密钥、管理员密码、医生密码

---

## 第四步：启动服务

```powershell
docker compose up -d
```

首次启动会自动下载镜像，大约 3-10 分钟（取决于网络）。

看到 `Container lab2fhir-backend Started` 和 `Container lab2fhir-frontend Started` 就成功了。

验证：

```powershell
docker compose ps
```

两个服务都是 `Up` 状态即可。

---

## 第五步：初始化数据库和用户

```powershell
docker compose exec backend python tools/seed_users.py
```

输出类似：

```
已创建默认用户:
  admin  / 你的密码  (pathology_staff - 病理科)
  doctor / 你的密码  (doctor - 科室医生)
```

---

## 第六步：打开防火墙

Windows Server 默认屏蔽 80 端口，需要放行。

**PowerShell（管理员）**：

```powershell
New-NetFirewallRule -DisplayName "HTTP 80" -Direction Inbound -Protocol TCP -LocalPort 80 -Action Allow
```

**云服务器额外步骤**（阿里云/腾讯云/AWS 等）：
登录云厂商控制台 → 安全组 → 入方向 → 添加规则 → 端口 `80`，来源 `0.0.0.0/0`

---

## 第七步：访问系统

浏览器打开：**`http://你的服务器IP`**

你会看到：

1. **首页仪表盘** — 统计数据、报告类型分布
2. **导入报告** — 拖入 PDF → 点"开始导入" → 输入 admin 账户登录
3. **报告列表** — 搜索、筛选已导入的报告

## 后续操作

### 导入 PDF 报告

1. 点击侧边栏"导入报告"
2. 拖入 PDF 文件
3. 点"开始导入"
4. 弹出登录框 → 输入 admin 账户 → 自动导入

### 查看日志

```powershell
docker compose logs -f backend    # 后端日志
docker compose logs -f frontend   # 前端日志
```

### 停止 / 重启

```powershell
docker compose stop               # 停止
docker compose start              # 启动
docker compose restart            # 重启
```

### 更新到最新版本

```powershell
cd C:\Lab2FHIR
git pull
docker compose up -d --build
```

### 忘记密码怎么办

```powershell
docker compose exec backend python -c "
import sys; sys.path.insert(0, 'src')
from src.db.models import SessionLocal
from src.db.user_models import User
from src.auth.security import hash_password

db = SessionLocal()
user = db.query(User).filter(User.username == 'admin').first()
user.password_hash = hash_password('新密码')
db.commit(); db.close()
print('密码已重置')
"
```

## 故障排查

| 问题 | 解决方案 |
|------|----------|
| 浏览器打不开 | 检查防火墙：`Get-NetFirewallRule -DisplayName "HTTP 80"` |
| `docker` 命令不识别 | 重启电脑，确保 Docker Desktop 在运行（右下角图标） |
| 页面空白 | 等待 30 秒刷新，首次启动较慢 |
| 导入报"网络错误" | `docker compose restart backend` |
| PDF 无法查看 | `docker compose logs backend` 查看错误日志 |
