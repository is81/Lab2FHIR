# Lab2FHIR 部署指南

> Windows Server 2019 · 手动部署 · 无需 Docker

---

## 部署流程总览

```
联网电脑（开发机）                    生产服务器（Windows Server 2019）
      │                                        │
      │ 下载安装包                              │
      ├─ Python 安装包 ────── U盘 ──────────→ ① 安装 Python
      ├─ Node.js 安装包 ───── U盘 ──────────→ ② 安装 Node.js
      ├─ 项目代码 ─────────── U盘 ──────────→ ③ 拷贝项目
      │                                        │
      │                                       ④ pip install
      │                                       ⑤ npm install + npm run build
      │                                       ⑥ 配置 .env
      │                                       ⑦ 初始化数据库 + 创建用户
      │                                       ⑧ 启动服务 + 开机自启
```

---

## 准备离线安装包（在联网电脑上操作）

### 下载 Python

浏览器打开：https://www.python.org/downloads/

下载 **Python 3.11.x**（Windows installer 64-bit），文件名类似 `python-3.11.9-amd64.exe`。

### 下载 Node.js

浏览器打开：https://nodejs.org/

下载 **LTS 版本**（Windows 64-bit .msi），文件名类似 `node-v20.x.x-x64.msi`。

### 拷贝项目代码

```powershell
# 把当前项目文件夹整个拷贝到 U 盘
# 可以删掉以下目录以减小体积（服务器上不需要）：
#   frontend/node_modules/
#   backend/src/uploads/*.pdf
#   .git/
```

### 下载前端依赖包（可选，让服务器装也行）

如果服务器完全断网，需要提前下载 npm 包：

```powershell
cd C:\Lab2FHIR\frontend
npm install   # 联网电脑上执行，会自动下载所有包到 node_modules/
```

然后把整个 `frontend/node_modules` 一起拷到 U 盘。

## 第一步：安装 Python（在服务器上）

### 1.1 安装

双击 U 盘里的 `python-3.11.x-amd64.exe`。

⚠️ **重要**：第一个界面勾选 **"Add Python to PATH"**（底部复选框），否则后面命令会找不到 python。

然后点 **Install Now**，等待完成。

### 1.2 验证

打开 PowerShell：

```powershell
python --version
```

显示 `Python 3.11.x` 即可。

---

## 第二步：安装 Node.js（在服务器上）

### 2.1 安装

双击 U 盘里的 `node-v20.x.x-x64.msi`，一路 Next 到底。

### 2.2 验证

```powershell
node --version
```

---

## 第三步：拷贝项目到服务器

```powershell
# 从 U 盘拷贝到 C 盘
Copy-Item -Recurse D:\Lab2FHIR C:\Lab2FHIR
cd C:\Lab2FHIR
```

---

## 第四步：安装后端依赖

```powershell
cd C:\Lab2FHIR\backend
pip install -r requirements.txt
```

如果报 `pip` 找不到：

```powershell
python -m pip install -r requirements.txt
```

---

## 第五步：安装前端依赖并构建

```powershell
cd C:\Lab2FHIR\frontend
npm install
npm run build
```

`npm install` 如果服务器不能联网，把联网电脑上已下载的 `node_modules` 文件夹拷贝过来即可。

构建完成后会生成 `frontend/dist/` 目录。前端 HTML 是静态文件，用 nginx 或 Python 自带的 http.server 托管即可。

---

## 第六步：配置密钥和密码

```powershell
cd C:\Lab2FHIR

# 生成随机 JWT 密钥
$jwtSecret = -join ((48..57) + (97..102) | Get-Random -Count 64 | ForEach-Object { [char]$_ })
Write-Host "JWT 密钥: $jwtSecret"
```

复制输出的字符串，然后创建配置文件：

```powershell
@"
LAB2FHIR_JWT_SECRET=粘贴生成的字符串
LAB2FHIR_ADMIN_PASSWORD=设置管理员密码
LAB2FHIR_DOCTOR_PASSWORD=设置医生密码
CORS_ORIGINS=*
"@ | Out-File -FilePath .env -Encoding UTF8
```

---

## 第七步：初始化数据库和用户

```powershell
cd C:\Lab2FHIR\backend
$env:PYTHONIOENCODING = "utf-8"
python tools/seed_users.py
```

如果报错找不到模块，先设置一下路径：

```powershell
$env:PYTHONIOENCODING = "utf-8"
$env:LAB2FHIR_JWT_SECRET = "temp-key"
$env:LAB2FHIR_ENV = "development"
python tools/seed_users.py
```

---

## 第八步：启动服务

需要开两个 PowerShell 窗口：

**窗口 1 — 后端**：

```powershell
cd C:\Lab2FHIR
$env:PYTHONIOENCODING = "utf-8"
python -m uvicorn backend.src.api.main:app --host 0.0.0.0 --port 8000
```

看到 `Application startup complete` 即启动成功。

**窗口 2 — 前端**：

```powershell
cd C:\Lab2FHIR\frontend
npx serve -s dist -l 80
```

`serve` 是一个简单的静态文件服务器。如果没装：

```powershell
npm install -g serve
```

---

## 第九步：打开防火墙

**PowerShell（管理员）**：

```powershell
New-NetFirewallRule -DisplayName "HTTP 80" -Direction Inbound -Protocol TCP -LocalPort 80 -Action Allow
New-NetFirewallRule -DisplayName "Backend 8000" -Direction Inbound -Protocol TCP -LocalPort 8000 -Action Allow
```

浏览器打开：**`http://服务器IP`**

## 第五步替代方案：用 nginx 托管前端

nginx 比 `serve` 更稳定，适合生产环境：

### 5.1 下载 nginx

在联网电脑下载：https://nginx.org/en/download.html

选 Stable version → Windows 版本（zip 包）。

### 5.2 安装

把 zip 解压到 `C:\nginx`。

### 5.3 配置

用记事本打开 `C:\nginx\conf\nginx.conf`，替换为：

```nginx
worker_processes 1;

events { worker_connections 1024; }

http {
    include mime.types;
    default_type application/octet-stream;
    sendfile on;
    keepalive_timeout 65;

    # gzip
    gzip on;
    gzip_types text/plain text/css application/json application/javascript text/xml;

    # 安全头
    add_header X-Content-Type-Options "nosniff";
    add_header X-Frame-Options "DENY";

    server {
        listen 80;
        server_name localhost;

        # 前端静态文件
        location / {
            root C:/Lab2FHIR/frontend/dist;
            index index.html;
            try_files $uri $uri/ /index.html;
        }

        # API 代理到后端
        location /api/ {
            proxy_pass http://127.0.0.1:8000;
            proxy_set_header Host $host;
            proxy_set_header X-Real-IP $remote_addr;
        }
    }
}
```

### 5.4 启动 nginx

用 nssm 注册为 Windows 服务（见下方"开机自启"）。

临时测试：双击 `C:\nginx\nginx.exe`。

## 开机自启

手动开两个窗口不方便，用 **nssm** 注册为 Windows 服务。

### 安装 nssm

在联网电脑下载：https://nssm.cc/download

解压 `nssm.exe` 到 `C:\Windows\System32\`。

### 注册后端服务

**PowerShell（管理员）**：

```powershell
nssm install Lab2FHIR-Backend "C:\Program Files\Python311\python.exe" "-m uvicorn backend.src.api.main:app --host 0.0.0.0 --port 8000"
nssm set Lab2FHIR-Backend AppDirectory "C:\Lab2FHIR"
nssm set Lab2FHIR-Backend AppEnvironmentExtra "PYTHONIOENCODING=utf-8"
nssm start Lab2FHIR-Backend
```

### 注册 nginx 服务

```powershell
nssm install Lab2FHIR-Nginx "C:\nginx\nginx.exe"
nssm set Lab2FHIR-Nginx AppDirectory "C:\nginx"
nssm start Lab2FHIR-Nginx
```

### 验证

```powershell
nssm status Lab2FHIR-Backend
nssm status Lab2FHIR-Nginx
```

两台都是 `SERVICE_RUNNING` 即可。重启服务器后会自动启动。

## 日常管理

```powershell
# 查看服务状态
nssm status Lab2FHIR-Backend
nssm status Lab2FHIR-Nginx

# 重启后端（比如改了配置后）
nssm restart Lab2FHIR-Backend

# 查看后端日志
Get-Content C:\Lab2FHIR\backend\lab2fhir.log -Tail 50
```

### 更新版本

在联网电脑拉取最新代码 → 重新构建前端 → 拷贝到服务器覆盖：

```powershell
# 服务器上
nssm stop Lab2FHIR-Backend
# 覆盖项目文件...
nssm start Lab2FHIR-Backend
```

### 忘记密码

```powershell
cd C:\Lab2FHIR\backend
$env:PYTHONIOENCODING = "utf-8"
python -c "
import sys; sys.path.insert(0, 'src')
from src.db.models import SessionLocal
from src.db.user_models import User
from src.auth.security import hash_password
db = SessionLocal()
user = db.query(User).filter(User.username == 'admin').first()
user.password_hash = hash_password('新密码')
db.commit(); db.close()
print('密码已重置')
"
```

## 故障排查

| 问题 | 解决方案 |
|------|----------|
| `python` 不识别 | 安装时没勾选"Add to PATH"，重新安装或手动添加 `C:\Python311\` 到系统 PATH |
| `pip` 安装报错 | `python -m pip install --upgrade pip` |
| 浏览器打不开 | 检查防火墙：`Get-NetFirewallRule -DisplayName "HTTP 80"` |
| nginx 启动失败 | `C:\nginx\logs\error.log` 查看错误日志 |
| 后端报 ImportError | `cd C:\Lab2FHIR\backend` 然后执行 |
| 中文乱码 | 检查是否设置了 `$env:PYTHONIOENCODING = "utf-8"` |
