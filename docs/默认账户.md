# 默认账户

> 首次部署通过 `seed_users.py` 创建账户。密码由你设置，不设置则随机生成。

## 创建账户

```bash
# 方式 1: 指定密码
LAB2FHIR_ADMIN_PASSWORD=我的密码 LAB2FHIR_DOCTOR_PASSWORD=医生密码 \
  PYTHONIOENCODING=utf-8 python tools/seed_users.py

# 方式 2: 随机生成（打印到终端，请保存）
PYTHONIOENCODING=utf-8 python tools/seed_users.py
```

| 用户名 | 角色 | 权限 |
|--------|------|------|
| `admin` | 病理科 | 导入+删除+查看 |
| `doctor` | 医生 | 仅查看 |

> 登录后可通过右上角按钮修改密码。
