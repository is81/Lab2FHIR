==============================================
  Lab2FHIR 部署命令清单
  Windows Server 2019 · 离线部署
==============================================

【准备工作 — 在联网电脑上】
  下载 python-3.11.9-amd64.exe     https://www.python.org/downloads/
  下载 node-v20.x.x-x64.msi        https://nodejs.org/
  下载 nginx-1.26.2.zip            https://nginx.org/en/download.html
  下载 nssm-2.24.zip               https://nssm.cc/download
  git clone https://github.com/is81/Lab2FHIR.git
  删掉 .git/ 和 backend/src/uploads/*.pdf（减小体积）
  如果服务器完全断网：提前 npm install（保留 node_modules/）


===== 服务器端 =====

【1. 安装软件】
  python-3.11.9-amd64.exe      勾选 "Add Python to PATH"
  node-v20.x.x-x64.msi
  解压 nginx-1.26.2.zip 到 C:\nginx
  解压 nssm-2.24.zip，把 nssm.exe 放到 C:\Windows\System32\

【2. 拷贝项目】
  Copy-Item -Recurse D:\Lab2FHIR C:\Lab2FHIR
  Copy-Item -Recurse D:\pip-packages C:\pip-packages
  Copy-Item -Recurse D:\node_modules C:\Lab2FHIR\frontend\node_modules   (如果有)

【3. 安装依赖】
  cd C:\Lab2FHIR\backend
  pip install --no-index --find-links=C:\pip-packages -r requirements.txt

  cd C:\Lab2FHIR\frontend
  npm install        (如果有网)
  或者拷 node_modules 跳过

【4. 构建前端】
  cd C:\Lab2FHIR\frontend
  npm run build

【5. 配置 nginx】
  用记事本编辑 C:\nginx\conf\nginx.conf，替换为以下内容：

  worker_processes 1;
  events { worker_connections 1024; }
  http {
      include mime.types;
      default_type application/octet-stream;
      sendfile on;
      keepalive_timeout 65;
      gzip on;
      gzip_types text/plain text/css application/json application/javascript text/xml;
      add_header X-Content-Type-Options "nosniff";
      add_header X-Frame-Options "DENY";
      server {
          listen 80;
          server_name localhost;
          location / {
              root C:/Lab2FHIR/frontend/dist;
              index index.html;
              try_files $uri $uri/ /index.html;
          }
          location /api/ {
              proxy_pass http://127.0.0.1:8000;
              proxy_set_header Host $host;
              proxy_set_header X-Real-IP $remote_addr;
          }
      }
  }

【6. 配置密钥和密码】
  cd C:\Lab2FHIR

  生成 JWT 密钥：
  $jwtSecret = -join ((48..57) + (97..102) | Get-Random -Count 64 | ForEach-Object { [char]$_ })
  Write-Host $jwtSecret

  创建 .env 文件（替换密钥和密码）：
  @"
  LAB2FHIR_JWT_SECRET=粘贴生成的密钥
  LAB2FHIR_ADMIN_PASSWORD=管理员密码
  LAB2FHIR_DOCTOR_PASSWORD=医生密码
  CORS_ORIGINS=*
  "@ | Out-File -FilePath .env -Encoding UTF8

【7. 初始化数据库和用户】
  cd C:\Lab2FHIR\backend
  $env:PYTHONIOENCODING = "utf-8"
  $env:LAB2FHIR_ENV = "development"
  python tools/seed_users.py

【8. 注册 Windows 服务（开机自启）】
  nssm install Lab2FHIR-Backend "C:\Program Files\Python311\python.exe" "-m uvicorn backend.src.api.main:app --host 0.0.0.0 --port 8000"
  nssm set Lab2FHIR-Backend AppDirectory "C:\Lab2FHIR"
  nssm set Lab2FHIR-Backend AppEnvironmentExtra "PYTHONIOENCODING=utf-8"
  nssm start Lab2FHIR-Backend

  nssm install Lab2FHIR-Nginx "C:\nginx\nginx.exe"
  nssm set Lab2FHIR-Nginx AppDirectory "C:\nginx"
  nssm start Lab2FHIR-Nginx

【9. 防火墙放行】
  New-NetFirewallRule -DisplayName "HTTP 80" -Direction Inbound -Protocol TCP -LocalPort 80 -Action Allow

【10. 验证】
  浏览器打开 http://服务器IP
  登录 admin 账户，导入 PDF 测试


===== 日常管理 =====

  查看服务状态:      nssm status Lab2FHIR-Backend
                     nssm status Lab2FHIR-Nginx
  重启后端:          nssm restart Lab2FHIR-Backend
  查看日志:          Get-Content C:\nginx\logs\error.log -Tail 20
                      Get-Content C:\nginx\logs\access.log -Tail 20

  忘记密码:          cd C:\Lab2FHIR\backend
                     $env:PYTHONIOENCODING = "utf-8"
                     python -c "import sys; sys.path.insert(0, 'src'); from src.db.models import SessionLocal; from src.db.user_models import User; from src.auth.security import hash_password; db = SessionLocal(); user = db.query(User).filter(User.username == 'admin').first(); user.password_hash = hash_password('新密码'); db.commit(); db.close(); print('OK')"


===== 更新版本 =====

  在联网电脑：
    cd C:\Lab2FHIR
    git pull
    cd frontend && npm install && npm run build

  拷到服务器覆盖：
    nssm stop Lab2FHIR-Backend
    Copy-Item -Recurse D:\Lab2FHIR C:\Lab2FHIR
    nssm start Lab2FHIR-Backend
