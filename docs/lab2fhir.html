Lab2FHIR

将非结构化化验单 / 病理 PDF 报告自动转换为国际标准 HL7 FHIR R4 格式，打通医疗数据壁垒

FHIR R4 LOINC 编码 MIT 开源

支持的报告类型

覆盖 4 大类病理 / 化验报告，自动识别、自动解析

类型包含识别方式
细胞学（妇科）DNA 倍体分析 + TCT 液基细胞学报告标题关键词
HPV 检测人乳头状瘤病毒基因分型"人乳头状瘤病毒"
细胞学（非妇科）穿刺 / 脱落 / 胸腹水细胞学"细胞学报告单"
常规病理胃镜病理 + 外科病理诊断"病理诊断报告单"

核心功能

从 PDF 上传到 FHIR 输出的完整链路

📄

PDF 自动解析

拖拽上传，自动提取患者信息、检测结果、诊断结论，支持批量导入

🔍

全文搜索

按姓名 / 病理号 / 诊断关键词搜索，FTS5 全文索引，毫秒级响应

🔒

角色权限

病理科管理员可导入报告，科室医生自由浏览查询，JWT 安全认证

⚡

FHIR R4 输出

生成标准 DiagnosticReport + Observation Bundle，含 LOINC 编码

👁

PDF 原文对照

三栏布局：解析结果 | PDF 原文 | FHIR JSON，可自定义缩放

⚙

轻量部署

SQLite + Python + nginx，无需 Docker、无需外部数据库

使用流程

三步完成从 PDF 到结构化数据的转换

1

上传 PDF

拖拽至导入页面，支持单份或批量

2

自动解析

识别报告类型，提取患者信息和诊断

3

查看结果

搜索、筛选，PDF 原文与 FHIR 对照
